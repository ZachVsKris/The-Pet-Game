# The Pet Game v2.2 Audit

## Gameplay changes

- Replaced player-facing “ranked higher” wording with “more popular” throughout the questions and reveals
- Reworked name and breed matchups to be closer and less obvious
- Preserved an Easy → Medium → Hard progression inside each round
- Added deterministic left/right randomization: the correct answer can appear on either side, while every player using the same daily date or seed sees the identical layout
- Removed the manual Next button
- Added a 2.4-second answer reveal followed by automatic advancement

## Daily and challenge modes

- Added a Daily Challenge keyed to the player’s calendar date
- Added random games, each with a shareable seed
- Added direct seed entry on the home screen
- Added URL support:
  - `?mode=daily&date=YYYY-MM-DD`
  - `?seed=PAWS42`
- Added a Challenge a Friend button on the results screen

## Validation

- 30 total questions
- 10 questions in each of three rounds
- Difficulty progression is retained even though questions shuffle within each difficulty tier
- Left/right placement is generated by a seeded pseudorandom function
- Next.js remains pinned to patched version 15.4.8

## v2.3 visual/mobile redesign
- Rebuilt the game board for small screens using two stacked choice cards sized to remain visible together on common phone viewports.
- Moved the short answer reveal into a mobile bottom sheet so it does not push either choice off-screen.
- Added safe-area support, compact mobile header/progress, and an extra breakpoint for narrow phones.
- Refined typography, spacing, shadows, round-specific accents, choice states, buttons, home screen, and results screen.
- Added visible A/B choice markers and stronger focus states.
- Added reduced-motion support.
